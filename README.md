# Ludo Royale

A browser-based board game collection built with pure HTML, CSS, and JavaScript.

## Features

- Ludo Royale game
- Snakes and Ladders game
- 2 to 4 players
- Single player vs AI
- Local multiplayer
- Animated realistic dice
- Player turns and active player indication
- Token movement with Ludo rules
- Safe zones, home path, captures, and winning system
- Responsive mobile-first design
- Scoreboard and restart button
- Sound effects and smooth animations

## Run Locally

Open `index.html` in any modern browser.
